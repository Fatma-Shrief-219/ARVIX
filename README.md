# ARVIX - Car Rental Website

ARVIX is a modern web application for car rentals, designed to provide a seamless experience for users to browse, book, and manage rental cars. The project is built using **HTML, CSS, JavaScript, Node.js**, and uses **SQL** for database management.

---

## Features

- Browse available cars for rent
- User registration and login
- Booking and reservation management
- Admin panel to manage cars and bookings
- Responsive and user-friendly design

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js
- **Database:** SQL
- **Tools:** VS Code, GitHub
